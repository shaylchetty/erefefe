Project Description


Our program will take in a picture of java code and show you its output as well as a summary of what the code does.

Inspiration

Have you ever seen a snippet of code, and not known what it does? CodeSnap is here to help!

What it does

Just take a picture, and upload it to our website! We will tell you what the output is, and give you a short summary of what the code does.

How it does it

  1. First, the program processes your image using optical character recognition.
  2. Then, we make syntactical adjustments to ensure that we can run your code.
  3. Next, we run your code on our server and return the output.
  4. Lastly, we display the output of the code, as well as a summary of the code provided by GPT3.

Applications

This can be used to grade coding assignments and tests with ease. It can be used to educate people about the code that they input.
